Sudoku Solver

Rules of Sudoku

  1. One number per cell
  2. One instance of any one number from 1 - 9 in a row
  3. One instance of any one number from 1 - 9 in a column
  4. One instance of any one number from 1 - 9 in a 3 x 3 box

Rules for this Solver

  1. Hit the generate button and your chosen difficulty to generate a random Sudoku puzzle of that difficulty

  2. Click on the cell you wish to change to alter it

  3. Click on the number you want to put in the selected cell

  4. When a number is selected it will also highlight the same number across the board

  5. Click the compare solution button to check your answer against the solution for a randomly generated puzzle

  6. Click the check button in order to check if an entered puzzle has one unique solution (you can use this on generated puzzles however using the same logic the puzzle will only be printed if there is only one unique solution)

  7. Click the solve button to print the solution to a generated or an entered puzzle

  8. You can change the grid size with the puzzle size buttons
